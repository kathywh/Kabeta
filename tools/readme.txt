The Beta simulator and assembler (bsim.jar) and its document (MIT6_004s09_lab_tool_bsim.pdf) are from MIT OCW.

The include file, kabeta.uasm, is derived from beta.uasm to adapt to Kabeta extension.

Run the following comand to start bsim (JAVA Runtime is needed):

    java -jar bsim.jar my_program.uasm &

The awk script, coe2mif.awk, is used to convert .coe file into .mif file.
